LizCoach is an extensive fitness and nutrition application applied in order to supervise one’s workout routines and diet. Some of the existent functionalities are workout plans, calorie counters and even personalizedplates and users preferences.

1. Features

User Data Management

- User Profile: The users can enter and store their own information such as name, age, weight, height and gender.
- Gender Change: Users can switch between male and female and the workouts and workout section gets updated.

Workout Routines

- Custom Workouts: Offers exercises grouped by limb used (hands, legs, stomache, whole body) and for male and female individuals.
- YouTube Integration: All the workouts are connected to a youtube link showing how it’s performed.

Calorie Calculator

- Product List: Preserving the format of the list of various food products and the calorie content, users can view them.
- Add Custom Products: Users can input new food products as they exist in their environment and the number of calories that the food products contain.
- Calorie Calculation: Based on the products, which are used in the preparation of the meals, users can determine the calorie content of the food.
- Total Calories: Users can display the total calorie intake dependent on the above calculations.

2. Installation

To run LizCoach, follow these steps:

1. Make sure that Python is among the programs installed in your machine.

2. Install the required packages using pip:
   This for installation of dependencies given in the requirements.txt file use command: pip install -r requirements.txt

3. Run the Application:
   python lizcoach.py

3. Usage

1. Launch the Application:
   Execute the application using the command specified in the installation part of this guide.

2. User Profile:

   Fill the basic demographic information about yourself (name, age, current weight, height, gender).
   It provides an interface to save data and allows creating a special profile for every user.

3. Workout Routines:

   To try a new workout or select a new workout routine go to the workout page.
   : * To watch the YouTube video of each workout simply click on the workout name.

4. Calorie Calculator:

   Check out a product list found on the website to analyze different foods and their caloric values.
   This should be done alongside considered addition of custom products should they be unavailable in the market.
   Go to the calorie calculator to decide what calorie rate of your meals is.
   Have a look at the overall calorie intake.

4. Dependencies

LizCoach relies on the following Python packages:

- customtkinter: For the design of a graphical user interface.
- PIL: For image processing.
- tkinter: These are mostly used for message boxes and the like in the Graphical User Interface [GUI].
- webbrowser: For opening YouTube links.

For a full list of dependencies use pip to list the contents of the requirements.txt file.
